Có 4 cách khai báo biến:

  1. var name
  2. let name
  3. const name
  4. không sử dụng gì cả chỉ cần tên

Khi nào thì sử dụng var,const,let

  1. từ khóa let sẽ khai báo biến chỉ sử dụng được trong block code. nó có tính chất private, khi khai báo ở trong 1 block code nó sẽ không thể được gọi lại ở ngoài khối code đó
  2. từ khóa const dùng để khai báo 1 biến mà giá trị của nó sẽ không thể thay đổi
  3. biến khai báo bởi vảr có tính hoisted, nó có thể sử dụng được ngay cả khi biến được khai báo sau đó.
